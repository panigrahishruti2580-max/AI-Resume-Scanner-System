# AI Resume Screening System

## Overview

The AI Resume Screening System is an NLP-powered application that automates the resume evaluation process by comparing candidate resumes against a given job description. The system extracts relevant skills, performs semantic matching, ranks candidates, and generates insights to support efficient recruitment decisions.

## Business Problem Statement

Recruiters often spend significant time manually reviewing resumes to identify suitable candidates. This process becomes challenging when handling a large volume of applications.

The objective of this project is to automate resume screening using Artificial Intelligence and Natural Language Processing (NLP) techniques, enabling faster, more accurate, and scalable candidate evaluation.

## Project Workflow

1. Upload Resume(s)
2. Enter Job Description
3. Extract Resume Content
4. Identify Skills and Candidate Information
5. Perform Semantic Matching
6. Calculate Match Score
7. Rank Candidates
8. Generate Insights and Reports
